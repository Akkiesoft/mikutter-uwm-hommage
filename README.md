mikutter-uwm-hommage
====================

# これなん？
penguin2716さんの画像投稿プラグイン（mikutter_update_with_media）のオマージュです。
mikutter_update_with_media用のGUIを書いてたんですが、いつの間にか単独動作可能になってたでござる。


# 特徴
* GUIがあります。
* twitter.rb他、追加のgemが不要です（日本語環境特有のバグを追ってたりしたら、実装した方が早い的な雰囲気に。）。
